# techi
A project to recreate a Techi website.
# Usage
```
npm run watch
```
# Use
* HTML5
* CSS3
* Modern JavaScript (ES2015+)
* Node.js
* Express
* MongoDB
* Nodemon
* OnChange
* Concat
